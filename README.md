# hnc
A hacker-news client using clojurescript, reagent, shadow-cljs.


## Setup And Run

#### Install dependencies
```shell
yarn install || npm install
```

#### Run dev server
```shell
yarn dev || npm run dev
```

#### Compile an optimized version

```shell
yarn release || npm run release
```
